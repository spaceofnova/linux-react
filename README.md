## Running Locally

### Prerequisites

- Node.js
- bun

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies using bun
4. Start the development server

```bash
git clone https://github.com/spaceofnova/linux-react.git
cd linux-react
bun install
bun dev
```
